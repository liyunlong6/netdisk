# NetDisk - 自托管网盘应用

## 项目简介

NetDisk 是一个轻量级的自托管网盘应用，基于 Nextcloud 功能总结开发。

## 技术栈

### 前端
- Vue 3 + Composition API
- Vite 6
- TypeScript 5
- Pinia (状态管理)
- Naive UI (UI 组件库)

### 后端
- Spring Boot 3
- MyBatis-Plus 3.5
- MySQL 8
- JWT (身份认证)

## 功能特性

- ✅ 用户注册/登录
- ✅ 文件上传/下载
- ✅ 文件夹管理
- ✅ 文件预览
- ✅ 分享链接
- ✅ 回收站

## 项目结构

```
NetDisk/
├── backend/       # Spring Boot 后端
├── frontend/      # Vue 3 前端
└── docker/       # Docker 配置
```
